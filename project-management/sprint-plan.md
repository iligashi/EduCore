# EduCore Sprint Plan

## Sprint 1 - Setup

- Create repository structure
- Configure frontend and backend TypeScript projects
- Configure MySQL, MongoDB, Docker, and environment files
- Add database schema and seed flow

## Sprint 2 - Authentication

- Register, login, refresh, logout, and `/me` APIs
- JWT middleware and role-based authorization
- Protected frontend routing and auth state

## Sprint 3 - DMS

- Student management
- Instructor management
- Class and schedule management
- Attendance recording and reports

## Sprint 4 - LMS

- Course creation
- Lesson CMS
- Assignment creation
- Submission upload
- Grading workflow

## Sprint 5 - Advanced Features

- Realtime notifications
- Advanced search
- CSV, JSON, and Excel import/export
- CMS pages and announcements
- Performance-based recommendations

## Sprint 6 - Finalization

- Responsive UI pass
- Swagger documentation
- Seed data
- Testing and bug fixing
- Final presentation and report
